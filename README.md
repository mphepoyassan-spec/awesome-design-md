# Awesome Design MD 🎨

A curated collection of AI-native `DESIGN.md` files inspired by world-class design systems.

Instead of generic UI, give your AI agent the visual philosophy, exact color tokens, and spacing principles of billion-dollar companies. Simply drop a file into your project and tell your AI to "build it like this."

## 🚀 How to Use

1. **Pick a Brand:** Choose a design language from the [brands/](./brands) directory.
2. **Drop into Root:** Copy the content into a file named `DESIGN.md` in your project root.
3. **Prompt your Agent:** Tell Cursor, Claude, or any AI: *"Build a [page/component] using the design principles in DESIGN.md."*
4. **Profit:** Get pixel-perfect, brand-accurate UI instantly.

## 🧩 What's inside each file?

- **Visual Philosophy:** The "why" behind the design.
- **Design Tokens:** Hex codes, spacing scales, and typography rules.
- **Component Patterns:** How buttons, cards, and navbars should behave.
- **AI Instructions:** Specific directives to help LLMs execute the style accurately.
