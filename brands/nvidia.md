# NVIDIA Design System (DESIGN.md)

NVIDIA's design system radiates "Technical Power." It is high-energy, high-contrast, and focused on showcasing performance and cutting-edge technology. It uses a signature green on black to create a "Matrix-like" but professional aesthetic.

## 1. Visual Philosophy
- **High Performance:** Visuals should feel fast, powerful, and data-rich.
- **Technical Energy:** Use of gradients and glowing accents to suggest light and processing power.
- **Extreme Contrast:** Stark black backgrounds make the "NVIDIA Green" pop.

## 2. Design Tokens

### Color Palette
- **Primary Brand:**
  - NVIDIA Green: `#76B900`
- **Backgrounds:**
  - True Black: `#000000`
  - Dark Slate: `#1A1A1A`
- **Text:**
  - Primary: `#FFFFFF`
  - Secondary: `#B2B2B2`
  - Green Accent: `#76B900`
- **Neutral:**
  - Dark Gray: `#333333`
  - Light Gray: `#E5E5E5`

### Typography
- **Primary Font:** NVIDIA Sans (or Industry/Inter)
- **Style:** Bold, clear, and modern.
- **Scale:**
  - **Hero:** 48px+, Weight 800, Uppercase accents.
  - **Headline:** 32px, Weight 700.
  - **Body:** 16px, Weight 400.

### Spacing & Layout
- **Base Unit:** 8px
- **Border Radius:**
  - `sharp`: 0px (For a more technical/industrial feel)
  - `standard`: 4px (Buttons, Cards)
- **Grid:** Geometric and structured.

### Effects
- **Glow:** `0 0 10px rgba(118, 185, 0, 0.4)` (Used for active states/highlights).
- **Gradients:** From `#76B900` to `#4D7A00` for depth.

## 3. Component Patterns

### Buttons
- **Primary:** Background `#76B900`, Black Text, Weight 700.
- **Outline:** Border `1px solid #76B900`, Green Text.

### Cards
- Dark Slate background (`#1A1A1A`), Green top border (2px), sharp corners.

## 4. AI Implementation Instructions
- **Theme:** Dark Mode by default.
- **Accents:** Use NVIDIA Green for call-to-actions, code snippets, and success states.
- **Vibe:** "Powerful, Industrial, Cutting-edge, Cinematic."
- **Graphics:** Favor high-quality renders and technical diagrams.
